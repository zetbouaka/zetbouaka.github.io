면접장에서 이런 문제를 풀게 시켰으면 어땠을까 싶은 아쉬움이 듭니다.

1번 문제

Josephus problem의 변형.

Josephus Problem의 풀이법(영어)

기본 Josephus Problem의 점화식은 다음과 같다.

f(n, k) = (f(n-1, k) + k) % n

우선, d = 1일 경우를 생각해보자.

f(n, k, j) = (f(n-1, k, j) + k + ?) % n

예전 Josephus에는 없었던 j의 존재 때문에 무엇인가를 추가적으로 더해줘야 한다. 대략적으로 마지막 움직임은 k + j * (n-1)이 될 것 같으니까

f(n, k, j) = (f(n-1, k, j) + k + j * (n-1)) % n

이 된다. 반시계방향일 경우도 마찬가지로 계산하면 됩니다.

이것도 귀찮으면 그냥 python-list를 써서 편하게 구현할 수도 있습니다. 대신 속도가 좀 낭비되겠지만(얼마나 낭비되는지는 귀찮아서 테스트 안 해 봄)

3번 문제

BOJ 2672와 거의 같은 방식으로 풀 수 있습니다.

단, 여기서는 1번 직사각형 하나만 겹치는 구간을 구하면 됩니다.

즉, 1번 직사각형에 겹치고, 2번 3번 직사각형에는 겹치지 않는 구간의 넓이를 모두 더해주면 됩니다.
